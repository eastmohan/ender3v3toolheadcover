# ender3v3toolheadcover
This is a new toolhead cover for my Ender 3 V3 3D printer that will have a camera and ring light and make the camera and light arm magnetic or otherwise detachable.
The arm with the two lights and the camera slides in and out of the main body. The ESP-32 C3 box attaches magneticaly to the side of the toolhead. The USB hub will be right by the printer's USB port to save space and keep the toolhead light.

I am making this so that I can monitor my prints remotely and also use the lighting to see it better. I may change the arm design if it is too front heavy. For fast prints, I will probably take off this mod so the toolhead is as fast as possible. After attaching this mod, run input shaper because this will change resonances and will change how your prints turn out.
